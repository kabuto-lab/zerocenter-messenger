# Релизы ME55

Готовые к запуску сборки. Скачал — распаковал — дабл-клик. Никаких компиляторов, ничего ставить не надо.

## Windows (x64)

📦 **[ME55-windows.zip](ME55-windows.zip)** — ~7.4 МБ

Что внутри архива:

| Файл | Что это |
|------|---------|
| `ME55AGUI.exe` | сам мессенджер (Windows GUI) |
| `start.bat` | запуск с правильными настройками (двойной клик) |
| `README.txt` | пошаговая инструкция на русском |
| `WHY_ME55.html` | объяснение что это вообще такое и зачем |

### Как использовать

1. Скачай `ME55-windows.zip` (кнопка **"View raw"** на странице файла в GitHub, или клик правой → "Сохранить как").
2. Распакуй в любую папку — например, на Рабочий стол.
3. Дабл-клик по **`start.bat`**.
4. Windows покажет SmartScreen-предупреждение про неподписанное приложение → **"Подробнее" → "Выполнить в любом случае"** (приложение open-source, мы пока не платим $200/год за code-signing-сертификат).
5. Откроется тёмное окно ME55 Messenger. В левом верхнем углу — твой Peer ID (длинная строка `12D3KooW...`). Клик по ней копирует в буфер.
6. Обменяйся Peer ID'ами с тем, с кем хочешь переписываться (через любой канал — Telegram, голос, листок бумаги).
7. В GUI: "+" возле Contacts → вставь Peer ID собеседника → дай имя → Add. Кликни по контакту, набирай сообщение, Enter.

Подробности и решения проблем — в `README.txt` внутри архива.

### Системные требования

- **Windows 10 / 11** (x64)
- Microsoft Edge WebView2 (предустановлен на Win10+; если нет — `start.bat` подскажет где скачать)
- ~50 МБ свободного места + интернет

### Что делает приложение при первом запуске

- Создаёт identity (Ed25519 keypair) в `%LOCALAPPDATA%\ME55\friend\`
- Подключается к публичной bootstrap-ноде (45.9.40.37) для входа в P2P-сеть
- Слушает на случайном порту
- Создаёт локальную базу сообщений `messages.db` (зашифрованную)

Никакого аккаунта, регистрации, телефона, email — ничего вводить не нужно. Identity — это просто файл на твоём компе. Удалишь файл — будешь другим человеком в сети.

## macOS / Linux

Сборки пока нет — собирается из исходников:
```bash
git clone https://github.com/kabuto-lab/zerocenter-messenger.git
cd zerocenter-messenger
cargo build --release --features gui
./target/release/ME55AGUI --profile me
```

Требования: Rust 1.75+, pkg-config, OpenSSL/Webkit dev-headers.
