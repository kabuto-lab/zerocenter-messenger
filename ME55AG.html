Data Flow: Message Sending

// 1. User composes message plaintext = "Hello" // 2. Encrypt with Double Ratchet ciphertext = ratchet.encrypt(plaintext, recipient_pubkey) // 3. Wrap in protocol envelope message = ProtocolMessage { to: recipient_pubkey_hash, from: sender_pubkey, payload: ciphertext, timestamp: now(), ttl: 7_days } // 4. Serialize (protobuf) + sign serialized = protobuf_encode(message) signature = ed25519_sign(serialized, sender_privkey) packet = (serialized, signature) // 5. Find recipient via DHT or store for offline if recipient_online: dht_route(packet, recipient_pubkey_hash) else: dht_store(packet, recipient_pubkey_hash, ttl=7_days) // Also cache at mutual friends (F2F) friend_cache(packet, recipient_pubkey_hash) // 6. Transport layer picks best available transport.send(packet, obfuscation=adaptive)

Network Topology

🕸️
Structured Overlay
Kademlia DHT

Each peer is a DHT node. XOR metric for routing. O(log n) lookup time. Redundant storage across k-closest nodes.

🔄
Gossip Propagation
Epidemic Broadcast

Fan-out to random peers for message dissemination. Eventually consistent. Resilient to partial network partitions.

🛡️
Friend-to-Friend
Trusted Subgraph

Direct connections to known contacts. Store encrypted messages for offline friends. Higher trust, lower latency.

🌉
Circuit Relays
NAT Traversal

Community-run relay nodes for peers behind restrictive NAT. Rotating selection, no single relay knows full path.

🔒 Security Model

Threat Actors

🔴 Nation-State Adversary

Global passive observer, controls ISPs, can compel key disclosure, physical device seizure, mass DPI.

Countermeasures:
E2EE with PFS, no metadata storage, obfuscated traffic, no central points, deniable authentication.
🟡 Local Censor

Blocks domains/IPs, DPI analysis, throttles encrypted traffic, injects RST packets.

Countermeasures:
Domain fronting, traffic mimicry, multi-transport fallback, P2P bootstrap through blocked networks.
🟡 Sybil Attacker

Floods network with malicious nodes, tries to eclipse victims, intercepts DHT lookups.

Countermeasures:
Proof-of-work for node ID generation, web-of-trust for peer validation, random peer selection.
🟢 Criminal Actor

Targets individual users, malware, phishing, physical device access.

Countermeasures:
Secure enclaves for keys, biometric auth, remote wipe, plausible deniability (hidden volumes).

⚠️ Risk Assessment

Technical Risks

🔴 Cryptographic Failure

Implementation bug breaks E2EE, or quantum computers break current crypto.

Mitigation:
Use only audited libraries (ring, libsignal). Formal verification for critical paths. Crypto agility for post-quantum upgrade.
🟡 Network Partition

DHT fragmented, users can't find each other. Bootstrap fails completely.

Mitigation:
Multiple bootstrap methods (DHT, blockchain, mesh). Aggressive peer caching. Urgent mode: Bluetooth local sync.
🟡 Metadata Leakage

Traffic analysis reveals social graph despite E2EE. Timing attacks.

Mitigation:
Constant-rate cover traffic. Mixnets for high-threat users. Randomized message delays.

Operational Risks

🔴 Developer Compromise

Core devs arrested, forced to insert backdoor, or devices seized.

Mitigation:
Multi-sig releases (5+ devs required). Anonymous development. Reproducible builds detect tampering. No single point of failure.
🟡 Supply Chain Attack

Dependency compromised (typosquatting, malicious crate).

Mitigation:
Vendored dependencies, checksum verification, minimal dependencies, audit all crates.
🟢 User Error

Key loss, phishing, screenshots of sensitive chats.

Mitigation:
Social recovery for keys. UI warnings for unverified contacts. Disappearing messages by default.

Legal & Political Risks

🔴 Protocol Criminalization

Using the messenger becomes illegal. Possession of software = crime.

Mitigation:
Plausible deniability (looks like standard app). Stealth mode (hidden icon). No central org to prosecute. Widespread use = political cost.
🟡 Infrastructure Siege

ISPs forced to block P2P traffic entirely. Deep packet inspection mandated.

Mitigation:
Mimic standard protocols (HTTPS, video). Domain fronting through major CDNs. Rapid protocol evolution (cats vs mice).
🛡️
Final Defense: If all else fails, the protocol is open source. Anyone can fork, anyone can run. The idea cannot be killed, only the instances. And instances are infinite.
